# 插入排序

```js
        function insertSort(arr) {
            let len = arr.length
            for (var i = 1; i < len; i++) { //大概思路首先把第一个元素当成一个开始的元素，从第二个开始插入，插入的原则首先把要插入的元素保存起来，防止丢失
                var temp = arr[i]//然受把他当作基准值和他之前的比较（他之前的都是排序好的）如果前面的比他还大那么就用前面的元素覆盖了他，然后一直覆盖，就会造成
                var j = i - 1//丢失一个最后的元素并且他要插入的位置和后面的元素值是相同的，别忘了我们最后一个元素已经保存了，所以当循环结束的时候我们把保存的值放入确定好的位置，重复操作就会排序完成
                while (j >= 0 && arr[j] > temp) { //为什么这里只能写temp却不能写arr[i]呢，因为表面上你觉得这两个值是一样的那么你就错了虽有明面上
                    console.log(temp, arr[i]) //arr[i]没有被重新赋值，但是注意因为这个是数组，arr[j+1]就等于a[i]下面如果条件成立arr[i]的值就会发生改变
                    arr[j + 1] = arr[j]//此时再进行比较就出错
                    j--
                }
                arr[j + 1] = temp
            }
            return arr
        }
        console.log(insertSort([3, 8, 1, 6, 9, 4]))
```
