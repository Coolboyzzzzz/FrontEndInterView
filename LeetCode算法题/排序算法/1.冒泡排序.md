# 1.冒泡排序

```js
var a = [3, 1, 2, 5, 6, 7, 8, 9, 1]
for (var i = 0; i < a.length - 1; i++) { //冒泡的总次数
    for (var j = 0; j < a.length - i - 1; j++) { //第二层循环是每次都少i个因为每运行一次外循环都会确定好一个位置
        if (a[j] > a[j + 1]) {//让相邻的作比较
            a[j] = a[j] + a[j + 1]//这里没有利用中间变量
            a[j + 1] = a[j] - a[j + 1]
            a[j] = a[j] - a[j + 1]
        }
    }
}
console.log(a)
//输出 [1, 1, 2, 3, 5, 6, 7, 8, 9]
```
