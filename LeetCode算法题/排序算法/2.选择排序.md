# 选择排序

```js
        var arr = [5, 3, 6, 4, 7, 9, 5, 1, 2, 3]
        for (var i = 0; i < arr.length - 1; i++) { //选择排序 
            var min = i //把第一个最小的索引复制
            for (var j = i + 1; j < arr.length; j++) {
                if (arr[min] > arr[j]) {
                    min = j //找出最小位置下标的索引
                }
            }
            arr[i] = arr[i] + arr[min]
            arr[min] = arr[i] - arr[min]
            arr[i] = arr[i] - arr[min]
```
